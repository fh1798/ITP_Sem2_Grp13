\subsection*{Title of the Project}
Fragrance Webshop

\subsection*{Problem Description}
A fictive small shop that specializes in fragrances and perfume wants to reach a wider audience of customers than possible in their small town. The plan is to do so using a webshop in order to sell to customers abroad.

\subsection*{Solution Description}
To find a solution for this problem, we want to create a website for said small store. This website should include a webshop using a database of the shippable fragrances, and all utilities needed to have them shipped.

\subsection*{Technical Environment}
The main work will happen in PHP and HTML. The project will utilize an SQL database running locally with the help of XAMPP. Collaboration on the actual code will happen via GitHub.
